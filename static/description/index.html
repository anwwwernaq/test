All in One sales order reports odoo app helps users to print sales day wise reports, payment report for customer invoice/sales, product sales summary reports, user wise sales details report, highest sales products report, top customer product report, sales day book report with particular date range for particular company in XLS and PDF format. Users can also print category wise sales order pdf reports and sale order excel reports for single or multiple sale orders.

Features

Sales Day Wise Report

User can print day wise sale order report.

User can print payment report for customer invoice of sale order.

Access Rights Configurations

Users with "Print Sales Reports" access rights can only print all sales orders report, If user wants to see salesperson on invoice payment report user must have "Salesperson on Invoice/Sales Payment Report" access.

Sales Day Wise Report

Sales Day Wise Report Menu

Under Sales > Advance Sales Reports user can see "Sales Day Wise Report" menu.

On clicking "Sales Day Wise Report" menu wizard will open here user can select "Start Date" and "End Date", user can also select "Companies" and print report.

Sales Day Wise PDF Report

On clicking "PRINT PDF" button user can see printed report in PDF format.

Sales Day Wise XLS Report

On clicking "PRINT XLS" button user can see generated report in XLS format.

Top Customer Product Report

Top Customer Product Report Menu

Under Sales > Advance Sales Reports user can see "Top Customer Product Report" menu.

On clicking "Top Customer Product Report" menu wizard will open here user can select "Report Type" as Basic or Compare, If user selects basic report type then enter "From Date", "To Date", "Number of Items", "Total Sale Amount" and "Companies" and print report.

If user selects compare report type then enter "From Date", "To Date", "Compare From Date", "Compare To Date", "Number of Items", "Total Sales Amount" and "Companies" and print report.

Top Customer Product PDF Report

On clicking "PRINT PDF" button user can see printed report in PDF format.

Top Customer Product XLS Report

On clicking "PRINT XLS" button user can see generated report in XLS format.

Sales Order Category Report

Sales Order Category Report Menu

Under "Print" menu after selecting sales orders in tree view user can see "Sale Order Category Report" menu, User can also print report for multiple sale orders.

User can also print report for single sale order by clicking "Sale Order Category Report" menu under print menu in form view.

Sales Order Category PDF Report

User can see printed sale order category report in PDF format.

Sale Order Excel Report

Sale Order Excel Report Menu

User can print sale order for multiple or single sale order by clicking "Generate Sale XLS Report" menu under "Action" menu.

On clicking "Generate Sale XLS Report" user can see generated sale order excel report.

If User selects multiple sale order then user can see all sale order report per sheet.
